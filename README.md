# Light-Sensitive-LED-mini-academic-IoT-project-
Description: One adafruit neopixel LED was used along with a light sensor. The light sensor reads in light data based on the light level of the external environment. That data is then passed into the microcontroller. The program manipulates (converts analog reading to color intensity level to change brightness).


This project was started and completed during April, 2026.
